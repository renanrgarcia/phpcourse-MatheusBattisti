# If structure 
``` php
if (condition) {
  // code
}
```

# Nested if structure
``` php
if (condition) {
  if (condition) {
    // code 1
  }
  // code 2
} 
```

# If/Else structure
``` php
if (condition) {
  // code
} else {
  // code
}
```

# Else If structure (Different conditions)
``` php
if (condition) {
  // code
} elseif (condition) {
  // code
} else {
  // code
}
```

# Switch structure
``` php
switch ($variable) {
  case $value1:
    // code
    break;
  case $value2:
    // code
    break;
  default:
    // code
    break;
}
```
