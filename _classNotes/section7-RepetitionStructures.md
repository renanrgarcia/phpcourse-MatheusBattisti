# Repetition Structures
## Extra
- count(array): Returns the number of elements in an array

## While 
``` php
while (condition) {
  // code
}
```

Ex: 
``` php
// Definition of counter $i
$i = 0;

// While loop
while ($i < 10) {
  echo $i;
  // Increment or decrement counter. Can be with other operators, like: $i *= 2;
  $i++;
}
```
